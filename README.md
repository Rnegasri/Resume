🌐 Personal Resume Website

This is my personal resume website built using HTML and CSS.
It showcases my education, skills, projects, internships, and achievements in a clean and professional format.

---

🚀 Features

- Responsive resume layout
- Profile photo integration
- Education displayed in table format
- Sections for skills, internships, projects, and leadership
- Clean and simple UI design

---

🛠️ Technologies Used

- HTML5
- CSS3
